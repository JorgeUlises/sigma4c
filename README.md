# Sigma4C
Es el entorno de desarrollo para el sigma4c.
```bash
#$ vagrant init centos/7
#Para iniciar la máquina
$ vagrant up
```
Se ejecuta la provisión:
```bash
$ vagrant provision
$ vagrant reload
$ vagrant ssh
```

¿MÁS FÁCIL?
-----------
```bash
$ ./install
```

¿Y LUEGO?
-----------
Después de que hayas iniciado toda la máquina, se puede ejecutar fácilmente:
```bash
$ ./run
```
